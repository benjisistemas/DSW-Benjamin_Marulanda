HTML es un markup language, lo que significa que está escrito con códigos que puede leer una persona sin que sea necesario compilarlo primero. En otras palabras, el texto en una página web está «marcado» con estos códigos para dar instrucciones al navegador web sobre cómo mostrar el texto. Estas etiquetas de marcado son las propias etiquetas HTML. Cuando escribes código en HTML, estás escribiendo etiquetas HTML. Todas las etiquetas HTML están hechas con un número de partes específica

las etiquetas entramos dentro de las etiquetas

Cuando queremos crear una página web, tenemos que tener en cuenta que el documento HTML que vamos a crear contendrá el esqueleto de la página, y la hoja de estilos es la que nos aportara la información de las formas, tamaños y los colores.
